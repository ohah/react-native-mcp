# React Native MCP Documentation

Documentation for the MCP (Model Context Protocol) server that automates and monitors React Native apps **with AI**. This site focuses on **how to use** the tools, not internal design for developers.

## What this documentation covers

- **Using MCP**: How to connect the React Native MCP server in Cursor, Claude Desktop, and GitHub Copilot CLI
- **Required tools**: Installing and verifying iOS (idb) and Android (adb)
- **E2E testing**: Writing YAML scenarios and running them automatically in GitHub Actions

## Quick start

1. Install the MCP server: `npm install -g @ohah/react-native-mcp-server` or `npx -y @ohah/react-native-mcp-server`
2. Add the server to your client (Cursor / Claude / Copilot)
3. Apply the Metro plugin and Babel preset to your React Native app
4. Install idb (iOS) and adb (Android), then run and connect your app

See [Install & connect](./guide/install) for step-by-step instructions.
