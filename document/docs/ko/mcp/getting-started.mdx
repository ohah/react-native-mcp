# 시작하기

1. **설치** — MCP 서버 설치 및 앱 설정: [설치 및 연결](./) 참고.
2. **등록** — 클라이언트(Cursor / Claude / Copilot)에 서버 등록: [MCP 사용법](./mcp-usage) 참고.
3. **실행** — Metro와 앱을 시뮬레이터/에뮬레이터에서 실행.
4. AI 클라이언트에서 MCP 도구(take_snapshot, tap, assert_text 등)로 앱을 제어·검증.

클라이언트별 상세 설정은 [MCP 사용법 — Cursor, Claude Desktop, Copilot](./mcp-usage)를 보세요.
