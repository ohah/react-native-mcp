# React Native MCP 문서

React Native 앱을 **AI와 함께** 자동화하고 모니터링하기 위한 MCP(Model Context Protocol) 서버 문서입니다. 개발자용 내부 설계가 아니라 **사용 방법**을 중심으로 안내합니다.

## 이 문서에서 다루는 내용

- **MCP 사용법**: Cursor, Claude Desktop, GitHub Copilot CLI에서 React Native MCP 서버를 연결하는 방법
- **필수 도구**: iOS(idb) / Android(adb) 설치 및 확인
- **E2E 테스트**: YAML 시나리오 작성과 GitHub Actions에서 자동 실행하는 방법

## 빠른 시작

1. MCP 서버 설치: `npm install -g @ohah/react-native-mcp-server` 또는 `npx -y @ohah/react-native-mcp-server`
2. 사용 중인 클라이언트(Cursor / Claude / Copilot)에 서버 설정 추가
3. React Native 앱에 Metro(cdp-interceptor)와 Babel 프리셋 적용
4. iOS는 idb, Android는 adb 설치 후 앱 실행·연결

자세한 단계는 [설치 및 연결](./guide/install)에서 확인하세요.
